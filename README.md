# Welcome to Kohtla-Järve Spordikeskus's new website
Project for kood/Garage48 Empowering Women Hackathon
created by Anastassia Tihhomirova (marketing), Nele-Liis Võhma (developing), Ilona Maharramova (idea owner)

How to start the program in local computer:
1. clone the project
2. open the project folder in terminal
3. "npm install" to install packages
4. "npm run build" to build project
5. "npm run preview" to run the site
